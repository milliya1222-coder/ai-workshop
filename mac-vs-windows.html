・ MAC 用戶補充

蘋果電腦（Mac）
用戶看這裡

課程裡 95% 的操作，Mac 和 Windows 一模一樣。
只有下面幾個小地方不一樣，先知道，就不會慌 🍎

這份是給用 Mac（蘋果電腦） 的妳。
跟 Windows 比起來，Mac 多了一個「會先問妳要不要允許」的安全機制 —— 聽起來嚴格，其實是 Mac 在保護妳。看懂下面這幾點，妳就能順順地跟上課堂。

// 1. 安裝時：Mac 可能會說「打不開」

📥

下載後第一次打開，被擋住怎麼辦

從 claude.com/download 下載後，打開檔案把 Claude 圖示拖進「應用程式（Applications）」資料夾。

第一次打開時，Mac 可能跳出「無法打開，因為來自未識別的開發者」——這很正常，照下面做就好：

  1. 在「應用程式」裡找到 Claude，按住 Control 鍵＋點一下（或用兩指點觸控板）
  2. 選 「打開」 → 再按一次 「打開」
  3. 之後就能正常使用了
💡 這一步只要做一次。Windows 通常沒有這個提示，是 Mac 特有的把關。

// 2. 打開 App：用 Spotlight 最快

🔍

不用每次去資料夾找

按 Cmd + 空白鍵 叫出搜尋框，打「Claude」按 Enter，就打開了。

（Windows 是按左下角開始功能表打字找；Mac 用 Spotlight 搜尋。）

// 3. ⭐ 最重要：Mac 會問「可以存取資料夾嗎」

🔐

跳出來問權限，按「允許」就好

當妳請 AI 在 Code 分頁動手（建資料夾、改文件）時，Mac 會跳出一個視窗問：

「Claude 想要取用你儲存在『桌面』中的檔案」

→ 按 「好」／「允許」，它才碰得到妳的檔案。這是 Mac 在替妳把關，不是出問題。

不小心按到「不允許」了？不用緊張，去開：
左上角 → 系統設定 → 隱私權與安全性 → 檔案與資料夾（或「完整磁碟取用權」） → 找到 Claude 把開關打開，重開 App 就好。
💡 Windows 通常不會這樣一個一個問資料夾權限 —— 這是 Mac 跟 Windows 最大的不同，也是 Mac 用戶最常卡的地方。先知道就不會慌。

// 4. 鍵盤：Mac 用 Cmd，不是 Ctrl

⌨️

複製貼上的按鍵不一樣

  • 複製：Cmd + C（Windows 是 Ctrl+C）
  • 貼上：Cmd + V
  • 全選：Cmd + A

課堂上我說「Ctrl+C」的時候，Mac 的妳就用 Cmd 那顆（⌘，在空白鍵旁邊）。

// 5. 卡住要傳畫面給我：Mac 的截圖鍵

📸

截圖丟群組，我才看得到妳卡在哪

  • 截一塊區域：Cmd + Shift + 4，再拖出範圍
  • 截整個螢幕：Cmd + Shift + 3

截好的圖會出現在桌面，直接拖進 LINE 傳給我就好。（Windows 是 Win+Shift+S。）

// 6. 懶得打字？Mac 內建「用講的」

🎙️

對著電腦說，它幫妳打字

  1. 到 → 系統設定 → 鍵盤 → 聽寫，把它打開
  2. 之後在打字的地方，按兩下地球鍵 🌐（或 Fn 鍵）就開始聽寫
  3. 對著麥克風講，講完再按一次結束

跟 AI 溝通用「說」的比打字快很多，尤其中文。
