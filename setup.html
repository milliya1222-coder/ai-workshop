課前準備 ・ SETUP

一起把工具裝起來

等一下妳會看到兩種「打開 AI」的樣子 ——
一種很專業、一種很友善。別緊張：妳只要裝友善的那個就好 💛

這份是給完全沒碰過程式的妳。先帶妳見識一下工程師在用的「終端機版」（看看就好、不用裝），
再一步步帶妳裝我們課堂真正要用的桌面 App。準備一杯茶 🍵，我們開始。

// 先見識一下：終端機版（看看就好，不用照做）

AI 工具最原始、最強的樣子，是一個黑黑的視窗，工程師在裡面打指令叫它做事 —— 大概長這樣：

PS C:\Users\you> irm https://claude.ai/install.ps1 | iex
…安裝中，跑出一堆文字…
PS C:\Users\you> claude
Welcome to Claude Code 🎉
💡 看起來有點硬齁？它真的很強，但對剛開始的我們來說，這個黑視窗容易讓人緊張。
好消息是 —— 妳完全不用碰它。同樣的能力，有一個用滑鼠點、看得懂的版本。
↓

// 我們課堂用這個就好：桌面 App

一樣的 AI、一樣的強，只是換成看得懂、用滑鼠點的樣子。跟著下面裝這一個就好。

// 開始前，先準備這些

  • 一台 電腦（Windows 或 Mac 都可以，不是手機或平板）
  • 一個 Google（Gmail）帳號
  • 一張 信用卡（用來訂閱 Claude Pro）
  • 大約 20 分鐘、不被打擾的時間

// 四個步驟

1

下載並安裝 Claude 桌面 App

  1. 打開瀏覽器（Chrome 或 Edge），在網址列輸入 claude.com/download 按 Enter
  2. 它會自動認出妳的電腦，點 下載（Download） 那顆按鈕
  3. 下載完打開那個安裝檔，照指示一路「下一步／同意」裝好
  4. 裝好後：Windows 從左下角開始功能表找「Claude」打開；Mac 從「應用程式」資料夾打開
💡 小提醒：這是「桌面 App」，跟瀏覽器網頁版的 Claude 長得有點像、但功能更多 —— 我們課堂要用的是這個 App。
2

用 Google 登入

  1. App 打開後，點 登入（Sign in）
  2. 選 「用 Google 繼續」 —— 這樣不用記新密碼
  3. 選妳的 Gmail、按同意，回到 App 看到聊天畫面就成功了
3

升級成 Pro 方案

  1. 點左下角妳的頭像 → Settings（設定） → Billing（帳單）
  2. 選 Upgrade（升級），挑 Pro 方案（約 US$20／月，台幣約 650）
  3. 用信用卡完成付款
⚠️ 一定要升級：免費版不能用我們課堂要學的 Cowork 和 Code，所以這步不能跳過。
💡 上完課想省錢，隨時回來這裡取消，不會被綁約。已經升級過的可以跳過這步。
4

認識三個分頁（這是重點）

桌面 App 上方有三個分頁，先認得它們在做什麼就好：

Chat・聊天

就是純聊天、問問題、請它幫妳寫東西。不會碰到妳電腦裡的檔案 —— 最安全、最熟悉的那一個。

Cowork・交辦

像把一件事「交辦」給它：它在雲端自己默默做，妳可以去忙別的，做完回來看成果。適合比較大、要花時間的任務。

Code・會動手（我們主要用這個）

這是「會動手」的那一個 —— 它看得到、也改得到妳電腦裡的檔案（幫妳建資料夾、改文件、做網頁）。每一步動手前都會先問過妳，妳同意它才動。我們課堂大部分時間在這裡。

// 一個會讓妳安心的設計：Ask 權限

在 Code 分頁，當 AI 要動手做事（建檔案、改東西）之前，它會先跳出來問妳「可以嗎？」，等妳按同意，它才會動。

  • 看得懂、是妳要的 → 按 同意 / Yes
  • 不確定、看不懂 → 先按 不要 / No，問我或在群組問，不會壞
  • 想停下來 → 按鍵盤 Esc 就停得下來
🛡 所以妳不會把電腦弄壞。它不會偷偷亂動，每一步都先問過妳。剛開始看不懂就先按 No，這是最安全的習慣。

✅ 完成檢查清單（都打勾就大功告成）
