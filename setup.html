課前準備 ・ SETUP

把 Claude Code 裝起來

第 1 次課之前，找個安靜的 20 分鐘，照著下面一步步做就好。
卡住了別緊張 —— 上課我們會一起把它弄順 💛

這份是給完全沒碰過程式的妳。每一步都拆得很細，看到英文也不用怕，照著點、照著貼就好。
準備一杯茶 🍵，我們開始。

// 開始前，先準備這些

  • 一台 Windows 電腦（不是手機或平板）
  • 一個 Google（Gmail）帳號
  • 一張 信用卡（用來訂閱 Claude）
  • 大約 20 分鐘、不被打擾的時間

// 四個步驟

1

註冊 Claude 帳號

  1. 打開瀏覽器（Chrome 或 Edge），在網址列輸入 claude.ai 按 Enter
  2. 點 Sign up（註冊），選 「用 Google 繼續」 —— 這樣不用記新密碼
  3. 進到聊天畫面後，試著打一句「你好」，看它有沒有回妳
2

升級成 Pro 方案

  1. 點左下角妳的頭像 → Settings（設定） → Billing（帳單）
  2. 選 Upgrade（升級），挑 Pro 方案（約 US$20／月，台幣約 650）
  3. 用信用卡完成付款
⚠️ 一定要升級：免費版不能使用 Claude Code，所以這一步不能跳過。
💡 上完課想省錢，可以隨時回到這裡取消，不會被綁約。
3

安裝 Claude Code

  1. 按鍵盤左下角的 Windows 鍵，直接打字輸入 PowerShell，按 Enter 打開它（會跳出一個深藍色視窗）
  2. 確認視窗左邊開頭是 PS C:\…（有 PS 就對了，代表妳在 PowerShell）
  3. 把下面這行複製起來，在藍色視窗裡按滑鼠右鍵就會貼上，然後按 Enter：
在 PowerShell 貼上irm https://claude.ai/install.ps1 | iex

畫面會跑出一些文字、停頓一下，這都正常 —— 等它跑完出現完成訊息就好。

🆘 卡住了怎麼辦：回到網頁版 claude.ai，把這句貼給它：
「我是 Windows 使用者、完全不懂程式，請一步步教我安裝 Claude Code，用繁體中文。」
照它說的做，出錯就截圖貼回去問它。它會很有耐心。
4

第一次啟動・登入

  1. 在同一個 PowerShell 視窗，輸入 claude 按 Enter
  2. 出現選單時，用方向鍵選「Claude account with subscription」按 Enter
  3. 瀏覽器會自動打開登入頁 → 點 同意／授權
  4. 回到視窗看到 「Welcome to Claude Code」 就成功了 🎉
  5. 打第一句話：「你好，我是第一次使用」，確認它會回應

想離開時，輸入 /exit 按 Enter 就好。

✅ 完成檢查清單（都打勾就大功告成）
